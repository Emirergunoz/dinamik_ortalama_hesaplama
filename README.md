# Dinamik Ortalama Hesaplama

Flutter ile geliştirilmiş, ders bilgilerine göre dinamik not ortalaması hesaplayan bir mobil uygulama.

## Özellikler
- Ders ekleme ve silme
- Harf notu ve kredi seçimi
- Anlık ortalama hesaplama
- Sade ve kullanıcı dostu arayüz

## Kullanılan Teknolojiler
- Flutter
- Dart

## Amaç
Bu proje, Flutter öğrenme sürecinde temel widget yapısını ve kullanıcı etkileşimini pekiştirmek amacıyla geliştirilmiştir.
